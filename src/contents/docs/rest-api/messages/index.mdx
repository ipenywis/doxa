## Overview

Messages represent short pieces of text submitted through the example REST API. This resource page groups the endpoint reference pages that belong to the same public API surface.

| Method | Path | Description |
| --- | --- | --- |
| `POST` | `/v1/messages` | Create a message record. |
| `GET` | `/v1/messages/{id}` | Retrieve a message record. |

## Schema

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `id` | `string` | Response only | Stable message identifier. |
| `body` | `string` | Yes | Message text submitted by the caller. |
| `metadata` | `object` | No | Optional caller-defined metadata. |
| `createdAt` | `string` | Response only | ISO timestamp for when the message was created. |
