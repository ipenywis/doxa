## Endpoint

```http
GET /v1/messages/{id}
```

Retrieves a message record by its identifier.

## Authentication

| Header | Required | Description |
| --- | --- | --- |
| `Authorization` | Yes | Bearer token used to authenticate the request. |

## Path Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `id` | `string` | Yes | Message identifier returned by `POST /v1/messages`. |

## Response

```json
{
  "id": "msg_01JY8Z7A3Q8M9K2W4N6P0R1S2T",
  "body": "Ship reference docs as a separate section.",
  "metadata": {
    "source": "template-example"
  },
  "createdAt": "2026-05-28T12:00:00.000Z"
}
```

## Errors

| Status | Code | Meaning |
| --- | --- | --- |
| `401` | `unauthorized` | The bearer token is missing or invalid. |
| `404` | `not_found` | No message exists for the supplied `id`. |
| `429` | `rate_limited` | The caller sent too many requests. |
