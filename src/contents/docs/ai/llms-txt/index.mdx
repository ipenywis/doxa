Doxa exposes AI-native text routes so external assistants and coding tools can discover, read, and cite your documentation.

## Routes

| Route            | Output                                                                           |
| :--------------- | :------------------------------------------------------------------------------- |
| `/llms.txt`      | A plain-text index of all docs pages with titles, URLs, and descriptions.        |
| `/llms-full.txt` | The full docs corpus concatenated into one plain-text document with source URLs. |

Both routes are generated at request time from `contentStore`, so they follow the active content adapter.

## How It Works

<Mermaid
  chart={`graph TD;

## Example Index Shape

```text showLineNumbers
# doxa

> Beautiful, fast documentation that stays out of your way.

## Docs

- [Overview](https://docs.example.com/overview): See what Doxa Docs can do.
- [Quickstart](https://docs.example.com/quickstart): Install and run Doxa.
```

## Example Full-Corpus Shape

```text showLineNumbers
# doxa

> Beautiful, fast documentation that stays out of your way.

---

## Overview
Source: https://docs.example.com/overview

Doxa is a production-ready documentation template...
```

<Note title="Set the site URL" type="warning">
  Set `VITE_SITE_URL` in production so generated source URLs use your real docs
  host instead of the local or fallback URL.
</Note>

## When To Use Each Route

| Need                                         | Use                                                         |
| :------------------------------------------- | :---------------------------------------------------------- |
| Let an AI tool discover available pages      | `/llms.txt`                                                 |
| Give an AI tool the full corpus in one fetch | `/llms-full.txt`                                            |
| Cite a specific docs page                    | URLs listed in either route                                 |
| Keep private docs private                    | Do not expose these routes publicly without access controls |
